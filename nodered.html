Low-Code Paradigm

¿Qué es Node-RED?

Desarrollado originalmente por IBM, es una herramienta de **programación visual** para conectar dispositivos de hardware, APIs y servicios en línea de forma sencilla.

Basado en Node.js

Aprovecha el modelo orientado a eventos y no bloqueante para alta eficiencia.

Programación de Flujo

Reduce la complejidad visualizando la lógica del sistema como un diagrama.

Interfaz de arrastrar y soltar (Drag and Drop)

Anatomía de un Flujo

Los Nodos

Son los bloques de construcción. Cada nodo realiza una tarea específica: recibir datos, procesarlos o enviarlos.

Los Flujos

Es el conjunto de nodos conectados. Representa el proceso completo de principio a fin.

Mensajes (msg)

El objeto JSON que viaja entre nodos. La propiedad principal es msg.payload.

Estructura típica de un Mensaje:

{
    "topic": "sensor/temperatura",
    "payload": 24.5,
    "qos": 1,
    "retain": false,
    "timestamp": 1698754123
}
